# 快捷面板

在酒馆界面左下角提供一个小型快捷操作面板，用于快速浏览和管理世界书条目与预设提示词。

## 功能

### 世界书管理

- **当前世界书**：展示当前激活的世界书（全局、角色卡绑定、聊天绑定），未打开角色卡时仅展示全局世界书
- **全局世界书**：展示所有世界书列表，标注全局启用状态
- 点击世界书名称可展开其条目列表，点击条目可将其添加到快捷区

### 预设管理

- **预设**：展示当前使用的预设的提示词列表，标注启用/禁用状态
- 点击提示词可将其添加到快捷区

### 快捷区（已选子项）

添加到快捷区的子项支持以下操作：

- **编辑内容**：点击子项名称可打开编辑器，修改世界书条目的 content 或预设提示词的 content（占位符类型不可编辑）
- **启用/禁用开关**：每个子项旁有 ON/OFF 开关，可实时切换启用状态
- **移除**：点击 x 移除该子项

快捷区的子项会持久化到浏览器 localStorage，重新加载脚本后自动恢复。

## 使用方式

脚本加载后，在酒馆界面左下角会出现一个 `WB` 小按钮，点击即可打开面板。点击面板外部区域可关闭面板。
